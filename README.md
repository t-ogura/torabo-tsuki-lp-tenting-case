# torabo-tsuki LP(S) ケース
torabo-tsuki LP(S)用の埋め込みチルトスタンド採用ケース

![ケース外観](img/main_img.JPG?raw=1)

## 注意点
- LEDインジケーターは見えなくなります。
- 右トラボの構成です。
- 組み立て済みであることを想定しています。
- 右のボトムプレートは下部を左ボトムプレートと同様に切り離す必要があります。

## 別途必要なもの
- チルトスタンド（後述）
- M2 x 5mm ネジ : 4本 （キット付属のネジだとギリギリ届かなくなります）
- オリジナルの電池ケース : 左右分 （[リンク](https://github.com/sekigon-gonnoc/torabo-tsuki-lp/tree/master/3d-models/STL)）

## 組み立て方法
torabo-tsuki LP(S)のボトムプレートのネジを外し、M2x5mmネジでケースとボトムプレートと共締めする。

## 対応チルトスタンド
55mm x 14mm x 4mm を想定（角は丸みがあること）  
下記写真は、遊舎工房様のチルトスタンドを採用 ([商品リンク](https://shop.yushakobo.jp/products/10730) )
![チルトスタンド搭載例](img/stand.JPG?raw=1)

## ケースの印刷について
家庭用3Dプリンターによる印刷は想定していません。きれいに印刷するためには工夫が必要かもしれません。  
JLCでスプレーペイントを利用する際には、備考欄に両面塗装する旨を記載してください。

## トッププレート及びボトムプレートについて
トッププレートとボトムプレートはキット付属の物の利用を想定しています。  
金属プレートを利用する際には、オリジナルのPCBファイルをstepに変換後、ケースに合わせて編集する必要があります。  
ボトムプレートを金属にすると、回路がショートしますので非推奨です。
